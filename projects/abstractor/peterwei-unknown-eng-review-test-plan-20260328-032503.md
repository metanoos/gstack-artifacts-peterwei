## Key Interactions to Verify
- Start a three-branch run, complete it, and confirm the saved artifact includes `schema_version`, typed `research` metadata, and top-level fitness summaries.
- Trigger one explicit self intervention during a run and confirm the next branch steps and saved artifact reflect the intervention.
- Score a saved artifact after the run and confirm the displayed live metrics match the persisted score output.
- Run with one branch failure and confirm the run still finishes with a clear status and a partially scoreable artifact.

## Edge Cases
- Legacy artifact without `schema_version` or `research` section.
- Artifact with empty interventions list.
- Artifact with one failed branch and two completed branches.
- Run where no intervention fires.
- Scoring path with branch collapse, where all branches produce near-identical outputs.

## Critical Paths
- `self-fork-lab` research run: preflight -> branch stepping -> optional intervention -> artifact write -> artifact score -> completion summary.
- Backward-compatibility path: load existing saved artifact -> apply default version behavior -> compute score without crashing.
- Intervention causality path: apply typed intervention -> next branch state changes -> persisted artifact records both the intervention and resulting branch summary.
