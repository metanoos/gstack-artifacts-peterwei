# Test Plan
Generated by /plan-eng-review on 2026-04-01
Branch: main
Repo: geometer-jones/the-big-learn

## Affected Pages/Routes
- `Codex guided-reading thread` — verify chapter-end save now carries learner-style artifacts and still preserves translation/response artifacts.
- `Claude guided-reading thread` — verify the same adaptive scaffolding instructions and save contract appear here too, not just in Codex.
- `the-big-learn progress-save --format json` — verify structured learner-style payloads, learner translation log payloads, and generated annotations persist together.
- `the-big-learn progress --format markdown|json` — verify saved progress survives the new top-level learner-style state without losing chapter status reporting.

## Key Interactions to Verify
- Finish a chapter, save `Personal Translation`, `Personal Response`, `learner_translation_log`, and `learner_style` in one payload.
- Re-save the same chapter with new learner-style evidence and verify merge behavior instead of duplicate or destructive overwrite.
- Switch from one work to another and verify global learner-style signals still exist while per-work overrides stay local.
- Install host skills and verify generated Codex and Claude prompt files stay in sync for shared adaptive-scaffolding instructions.

## Edge Cases
- Invalid `learner_style` payload shape, wrong types, unknown scopes, empty evidence.
- `learner_translation_log` present without explicit `personal_translation_en`, should derive the chapter translation deterministically.
- Explicit `personal_translation_en` plus `learner_translation_log`, explicit chapter translation should win.
- Re-saving a line already present in `learner_translation_log`, latest entry should replace by `line_id` without duplicating the line.
- Empty or missing learner-style payload, should not erase existing global profile accidentally.

## Critical Paths
- `Host thread -> progress-save -> reading-progress.json -> reread` must preserve adaptive state and chapter artifacts together.
- `Shared prompt source -> generated Codex skill + generated Claude skill -> host install` must keep both hosts behaviorally aligned.
- `Raw-source chapter with generated annotations -> chapter-end save` must continue saving `generated_annotations` correctly alongside the new adaptive payloads.
