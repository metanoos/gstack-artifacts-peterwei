## Edge Cases
- Artifact file missing `schema_version`, wrong `artifact_kind`, missing `branches`, missing selected branch, empty vector, invalid slot.
- Legacy list-style vector payload vs object payload vs multi-branch payload.
- Output directory does not exist yet.
- Output directory creation fails.
- Self-vector seed embedding dimension matches target vs requires projection.
- Two call sites using the shared self-vector init helper produce identical vectors for the same backend/seed inputs.
