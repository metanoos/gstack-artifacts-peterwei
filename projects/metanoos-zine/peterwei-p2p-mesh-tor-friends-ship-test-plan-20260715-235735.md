## Edge Cases
- An automation recipe bound to another or missing workspace cannot run or be scheduled in the current workspace.
- A missing provider, missing signing key, unavailable home relay, and failed external relay publish each surface an actionable failure.
- Shielded paths and unrelated mounted scopes do not enter scheduled-run context or replay follow state.
- Corrupt local recipe/lens storage fails closed to safe defaults.
- Windows verification launches npm through the command shell while other child processes remain direct.
